# 麻酔危機管理エイド（Crisis Aid）

離島・単独麻酔（バックアップのない環境）向けの、全身麻酔中の急変に対する
意思決定支援（cognitive aid）。体重と成人/小児を入れると投与量を自動計算します。

**▶ 使う（公開URL）:** https://nobumitsu-3141.github.io/crisis-aid/

- インターネット不要のオフライン動作（初回アクセス後）
- iPhone/iPad: Safariで開く →「共有」→「ホーム画面に追加」でアプリ化
- Android: Chrome →「アプリをインストール」

## 収録（9シナリオ）
LAST／悪性高熱（MH）／周術期アナフィラキシー／CICO・eFONA／大量出血（MTP）／
高カリウム血症／徐脈・心停止（ACLS＋小児PALS）／重篤な気管支痙攣／新生児蘇生（NCPR）

体重連動の投与量・反復投与タイマー・クイック検索・印刷/PDF。
新生児・小児は日本（JRC/NCPR）主・国際（AHA/NRP）併記。各タブに根拠ガイドラインを明記。

## ⚠ 注意
本ツールは公表ガイドラインに基づく意思決定支援であり、臨床判断・最新の各ガイドライン・
施設プロトコル・添付文書での確認に代わるものではありません。投与量・製剤規格は使用前に
必ず照合してください。緊急時はまず応援要請を。
